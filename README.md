# ✈️ Proyecto_FPFC — Planificación y Optimización de Vuelos

Proyecto académico desarrollado para el curso **Programación Funcional y Concurrente (2024-I)**.

Implementa un sistema para **planificar y optimizar itinerarios de vuelo**, aplicando técnicas de **programación funcional, concurrencia y optimización algorítmica** para encontrar asignaciones eficientes de rutas, tiempos y recursos.

---

## 🎯 Objetivo

Diseñar un planificador que:

- Procese información de vuelos/rutas
- Evalúe restricciones (tiempo, disponibilidad, conexiones, etc.)
- Genere asignaciones válidas
- Optimice costos/tiempos totales
- Ejecute cálculos de forma concurrente para mejorar rendimiento

---

## 🧠 Enfoque técnico

El proyecto aplica:

- Programación funcional (inmutabilidad, funciones puras, composición)
- Concurrencia/paralelismo para procesamiento de múltiples vuelos
- Modelado de problemas como grafos/estados
- Algoritmos de búsqueda y optimización
- Separación clara entre lógica de negocio y ejecución

---

## 🛠️ Tecnologías

- Scala
- Paradigma funcional
- Concurrencia / paralelismo
- Estructuras de datos y grafos
